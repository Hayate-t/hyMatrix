# hyMatrix

シンプルな行列計算を目的とした、C++ライブラリです。
Eigen等の高度なライブラリを用いずに基本的な線形代数処理を行います。

## Version

- 0.1.0 (Latest) 2026/3/29

現段階では開発中となっています。
今後、ベクトルや回転行列などの機能を追加予定です。

## 機能

- 行列の和・差
- 行列の積
- スケール（実数倍）
- 連立方程式の解の算出
- 単位行列の生成
- 転置行列の生成
- 逆行列の生成
- 階数

## セットアップ
全ての機能が一つのヘッダファイルにまとめられているため、簡単に導入できます。

1. hyMatrix.hppを任意のファイルに置く
2. includeする
   `#include "hyMatrix.hpp"` （同一ディレクトリの場合）

## 使用方法

hyMatrix全体は、`linalg`名前空間に定義されています。

行列の全ての成分はdouble型を用いています。

### 行列の定義・値の代入
```cpp
linalg::Matrix M(2, 3); //2x3行列を定義

//各成分ごと値を代入
M(0, 0) = 1.0;
M(0, 1) = 4.0;
M(0, 2) = 2.0;
M(1, 0) = -2.0;
M(1, 1) = 5.0;
M(1, 2) = -3.0;
```

行列を定義したとき、デフォルトの値は0.0となります。

### 値の参照

```cpp
std::cout << M(1, 0);    //出力: -2
double i = M(0, 1) * 2;  //8.0
```

※行列のインデックス番号が0から始まることに注意が必要です。

例：九九の生成

```cpp
linalg::Matrix M(9, 9);
for (size_t i = 0; i < 9; i++) {
  for (size_t j = 0; j < 9; j++) {
    M(i, j) = (i + 1) * (j + 1);
  }
}

for (size_t i = 0; i < 9; i++) {
  for (size_t j = 0; j < 9; j++) {
    std::cout << M(i, j) << " ";
  }
  std::cout << std::endl;
}
std::cout << std::endl;
```

### 行列の加算・減算

`+`演算子や`-`演算子を用います。

```cpp
linalg::Matrix A(2, 2);
linalg::Matrix B(2, 2);

/**
  A,Bの初期化処理
**/

linalg::Matrix C = A + B;
linalg::Matrix D = A - B;
```

加算・減算は行列の型が一致していない場合、エラーになります。

### スケール

行列とdouble型の数（scaler）に対して`*`演算子を用いることで、行列の全ての成分をスケールすることができます。

```cpp

linalg::Matrix M(2, 2);

//各成分ごと値を代入
M(0, 0) = 2.0;
M(0, 1) = 3.5;
M(1, 0) = -2.0;
M(1, 1) = 6.0;

double k = 4.0;

linalg::Matrix X = M * k;
std::cout << X(1, 0);      //出力: -8
```

また、`M * k`、`k * M`の両方の記法に対応しています。

### 行列の積

2つの行列に対して`*`演算子を使うことで、行列の積を計算できます。しかし、演算子の左側の行列の行数と、右側の行列の列数が一致しない場合エラーになります。

```cpp

linalg::Matrix A(2, 3);
linalg::Matrix B(3, 1);

/**
  A,Bの初期化処理
**/

linalg::Matrix X = A * B;  //Xは2x1行列になる
```
仕様上、` A * B * C `とした場合、その結果は

$$
(AB)C
$$

に等しくなります。行列の積を複数回適用する場合は`()`を付けて順番を明記することを推奨します。

### 転置行列

```cpp
linalg::Matrix tM = M.transpose();
```

### 単位行列
静的メソッドです。任意の次数の単位行列を生成します。
```cpp
linalg::Matrix I = linalg::Matrix::identity(3);  //3次正方行列
```


### 逆行列
静的メソッドです。Gauss-Jordan法により逆行列を生成します。

```cpp
linalg::Matrix X = linalg::Matrix::inverse(M);
```

行列が正方行列でない場合や、正則でなく逆行列が存在しない場合はエラーになります。

### 連立方程式の解

静的メソッドです。任意の行列と列ベクトル（列数が1の行列）を用いて連立方程式を解くことができます。返り値は列数が1の行列です。

例:

```math
\begin{cases}
3x + 2y = 1 \\
x + y = 4
\end{cases}
```


```cpp
linalg::Matrix A(2, 2);
linalg::Matrix b(2, 1);

A(0, 0) = 3;
A(0, 1) = 2;
A(1, 0) = 1;
A(1, 1) = 1;

b(0, 0) = 1;
b(1, 0) = 4;

linalg::Matrix x = linalg::Matrix::solve(A, b);

std::cout << x(0, 0) << " : " << x(1, 0) << std::endl;  //結果: -7 : 11
```

第1引数が正方行列でなかった場合、第2引数が列ベクトルでなかった場合や、行列の行数と列ベクトルの要素数の不一致によってエラーになります。
Ver0.1.0時点では、方程式が解なしや無数に解があった場合もエラーになります。

### 階数

静的メソッドです。行列の階数を計算します。

```cpp
linalg::Matrix A(3, 3);
A(0, 0) = 1;
A(0, 1) = 2;
A(0, 2) = 1;
A(1, 0) = 2;
A(1, 1) = 4;
A(1, 2) = 2;
A(2, 0) = 1;
A(2, 1) = 1;
A(2, 2) = 0;

std::cout << linalg::Matrix::rank(A);  //出力: 2
```
